💻 Windows Terminal
---
Custom Windows Terminal configuration with personalized styling and behavior (catppuccin mocha themed).

---

⚠️ Note
---
«Some settings may not work exactly the same on your system.
Make sure to:»

- Use the latest version of Windows Terminal  
- Install required fonts : [jetbrains mono Nerd](https://www.nerdfonts.com/font-downloads)

---

📂 Files Included

- "settings.json" → Main configuration file  

---

⚙️ Installation

1. Open Windows Terminal  

2. Go to Settings → Open JSON file  

3. Replace the existing content with the downloaded config  

4. Save the file and restart Windows Terminal  

---

💡 Tips

«If changes don’t apply:»

- Restart Windows Terminal  
- Check font or theme availability
