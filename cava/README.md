#### ⚙️ Installation (Windows)

1. Install [CAVA](https://github.com/karlstav/cava)  
-     winget install karlstav.cava

2. Download(copy) my config file from  
   → [here](./config)

3. Place the file in:  
-     C:\Users\YourUsername\\.config\cava\

4. Replace existing config (if any)  

---

💡 __*Tips*__

«If bars are not moving:»

- Check your audio output source  
- Try changing input method in config  
- Restart CAVA / terminal
